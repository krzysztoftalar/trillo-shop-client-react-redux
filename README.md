## Getting Started

### Prerequisites

-  Node.js

### Installation

1. **Navigate into your project directory and run the following command to download packages.**

```shell
npm install
```
      
2. **Start the development environment.**

```shell
npm start
```

Your site is now running at `http://localhost:3000`
